# Pogo Flashing Fixture

This is a generic flashing tool for multiple PCB's. 

The core concept is that this fixture accepts 3d printed or machined inserts that follow a standarized hole pattern.
So, all you have to do is bolt in your PCB holder insert (usually copes to the OD of the board, but can use any method you choose), bolt in your pogo pin insert, connect your electrical equipment to your pogo insert via whatever method you choose, press and do whatever operation is necessary.

This is designed to use 8mm shafts and LM8UU linear ball bearings, which are 'economy of scale' cheap on Amazon (due to being an extremely common consumer 3d printer part).
